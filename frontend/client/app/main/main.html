Twin Peaks - Fullstack Starter Kit

Twin Peaks is cleverly named after (a meeting room in the Arup San Fransisco office) its two main components: frontend and backend servers.

twinpeaks-frontend
https://github.com/ArupAUS/twinpeaks-frontend

This is an AngularJS application running on NodeJS with ExpressJS. You can use this repository alone, or match it with the twinpeaks-backend repository to leverage a RESTfull API in your application. We reverse proxy requests on /rest in the frontend application to the RESTfull API.

twinpeaks-backend
https://github.com/ArupAUS/twinpeaks-backend

This is a Django Rest Framework RESTfull API that can support your frontend application. If the backend server is running you will see a list of available API routes below:
